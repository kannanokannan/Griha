# griha v0.1.0

Privacy-first, distributed home AI agent framework on Raspberry Pi SBCs.
Sibling to Sthala. Part of the ContextOps ecosystem.

Core principle: LLMs extract and narrate. Code verifies and executes. Always.

Stack: Python · NATS · LiteLLM · Pydantic · FastAPI · Home Assistant · WireGuard · Docker
